# 소수찾기

### 문제설명
- 1부터 입력받은 숫자 n 사이에 있는 소수의 개수를 반환하는 함수, solution을 만들어 보세요.

소수는 1과 자기 자신으로만 나누어지는 수를 의미합니다.
(1은 소수가 아닙니다.)


### 제한조건

- n은 2이상 1000000이하의 자연수입니다.

### 예시

|n|result|
|:---:|:---:|
|10|4|
|5|3|


### 문제풀이

반복문을 이용하여 n을 2...squt(n)을 차례대로 나눠 나머지가 있는지 확인한다.

또는  에라토스테네스의 체를 활용하여 푸는 방법도 있다.
