# 직사각형 별찍기

### 문제설명
- 이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다.
- 별(*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

### 제한조건

- n과 m은 각각 1000 이하인 자연수입니다.


### 예시

- 입력
```
5 3
```
- 출력
```
*****
*****
*****
```
### 문제풀이

단순 구현 문제로 ((n * "*") + \n) * m 을 구현하면 된다.

구현방법으로 나는 이중 for in 문을 사용했고.

다른 방법으로는 한줄을 array를 만들어 m만큼 출력하는 방법도 있다.
